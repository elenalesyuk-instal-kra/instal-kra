# **Instal-Kra — Commercial Web System**

A production-ready **commercial web system** for a company in the plumbing & heating industry.  
The project was developed, structured and deployed **end-to-end by a single developer**, with a strong focus on **system structure, integration, deployment and maintainability**, alongside development tasks.

---

## **🗓 Project Timeline**

**October 2025 – December 2025**

---

## **🌍 System Overview**

Instal-Kra is a **static commercial web system** designed to present company services, product categories, pricing information and partner content in a clear and structured way.

The system ensures:

- **Stable navigation**
- **Multilingual content handling**
- **Consistent behavior across devices and screen sizes**

The solution was built as a **production system**, considering deployment, operation, maintenance and future content updates.

---

## **🧩 System Architecture & Structure**

- **Responsive header** with structured navigation and pricing access  
- **Multilingual system** (Ukrainian ↔ Polish) with dynamic content switching  
- **Modular page sections**:
  - Services  
  - Partners  
  - Equipment  
  - Company  
  - Contacts  
- **Data-driven UI components**:
  - sliders
  - pricing cards  
  - generated from a structured local dataset  
- **Footer** with contact details, location and external links  

The system architecture separates **data, structure and presentation**, which allows easy maintenance and scalability.

---

## **📐 Responsive & Device Support**

The system is fully responsive and optimized for the following environments:

- **Mobile** — 390px  
- **Tablet** — 768px  
- **Small laptop** — 1024px  
- **Desktop** — 1440px  
- **Large screens** — 1920px  

All breakpoints were tested to ensure **layout stability and usability**.
